### 🗂️ 이미지 탐지 모델 사용 데이터셋
- 본 프로젝트에서는 이미지 기반 마약 객체 탐지 모델 구축을 위해 Roboflow에서 제공하는 `drugs-detection` 데이터셋을 YOLOv8 형식으로 다운로드하여 활용함. 데이터셋은 학습(train), 검증(valid), 평가(test) 데이터로 분할되어 있으며, 각 이미지는 객체의 위치 정보를 포함한 Bounding Box 라벨 파일과 함께 구성됨.
- 탐지 대상 클래스는 `Cocaine`, `Heroin`, `Marijuana`, `Shrooms`로 구성함. 본 이미지 탐지 모델은 단순히 이미지 전체가 마약 관련 이미지인지 분류하는 것이 아니라, 게시물 이미지 내에 존재하는 마약 객체의 위치와 종류를 동시에 탐지하는 것을 목표로 함.

### 🤖 이미지 탐지 모델 사용 알고리즘
본 프로젝트에서는 이미지 내 마약 객체의 위치와 클래스를 동시에 탐지하기 위해 YOLOv8 기반 객체 탐지 알고리즘을 사용함. YOLOv8은 입력 이미지에서 시각적 특징을 추출하는 Backbone, 다양한 크기의 객체를 탐지하기 위한 특징 결합 구조인 Neck, 그리고 객체의 Bounding Box, 클래스, Confidence를 예측하는 Detection Head로 구성됨.

YOLOv8은 한 번의 추론 과정에서 이미지 내 객체의 위치와 클래스를 동시에 예측할 수 있으므로, SNS 게시물 이미지처럼 객체의 크기, 촬영 각도, 배경 환경이 다양한 데이터에 적합함. 본 모델은 Roboflow에서 다운로드한 YOLOv8 형식 데이터셋을 기반으로 학습을 수행하였으며, 평가 과정에서는 Precision, Recall, mAP50, mAP50-95 및 클래스별 성능 지표를 활용하여 이미지 탐지 성능을 확인함.
